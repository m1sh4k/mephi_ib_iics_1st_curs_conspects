## Ячейка памяти с произвольной выборкой статического типа

![[SRAM_Cell_(6_Transistors).svg]]

Static Random Access Memory — **SRAM**

$M_{2}$ и $M_{4}$ выполняют балластную функцию, защищают от короткого замыкания.

$M_{5}$ и $M_{6}$ работают в ключевом режиме, соединяют с прямой и инверсной шинами данных.

Недостатки
- Много транзисторов
- Энергозависима

Преимущества
- Высокая скорость
- Та же технология что и для изготовления логических элементов

## Одно-транзисторная ячейка памяти с произвольной выборкой

Dynamic Random Access Memory

![[odnotrans_dram.drawio.svg]]
$RC=\tau$

Конденсатора как такового в схеме НЕТ — так изображена паразитная емкость транзистора.

Недостатки:
- Нужно успеть считать пока "конденсатор" не разрядится, записать обратно
- Требует постоянного цикла чтения, записи — **регенерации информации**

Преимущества:
- Мало транзисторов

Бит четность с вероятностью 50% позволяет определить ошибку передачи.
