## Спидран по Бесконечно малым функциям

$\beta(x)$ — б.м. при $x\to a$ если $\lim_{ x \to a }\beta(x)=0$

Бесконечно малая — если стремится к 0 в точке.

Бесконечно малые ...
- Эквивалентные ($\alpha \sim \beta$)
  $\lim_{ x \to a } \frac{\alpha(x)}{\beta(x)}=1$
- Одного порядка 
  $\exists \lim_{ x \to a } \frac{\alpha(x)}{\beta(x)}\neq0$
- Более высокого порядка (быстрее стремится к 0)
  $\alpha \in o(\beta)$
  $\lim_{ x \to a } \frac{\alpha(x)}{\beta(x)}=0$ 
  
### Свойства

1. $o(\beta)\pm o(\beta)=o(\beta)$
2. $c\cdot o(\beta)= o(\beta)$
3. $o(c\beta)=o(\beta)$ при $c \neq0$
4. $\beta^{n}o(\beta^{m})=o(\beta^{m+n})$
5. $o(\beta^{n})=o(\beta^{m})$ при $n \geq m$
6. $o(\beta+o(\beta))=o(\beta)$
7. $\alpha \sim \beta \Longleftrightarrow \alpha=\beta+o(\beta)$
8. $o(o(\beta))=o(\beta)$
   
#### Докво 1

$o(\beta)+o(\beta)=o(\beta)$

$\alpha_{1}=o(\beta)$
$\lim_{ x \to a } \frac{\alpha_{1}(x)}{\beta(x)}=0$

$\alpha_{2}=o(\beta)$
$\lim_{ x \to a } \frac{\alpha_{2}(x)}{\beta(x)}=0$

$\lim_{ x \to a } \frac{{\alpha_{1}(x)+\alpha_{2}(x)}}{\beta(x)}=\cancelto{ \ 0 }{ \lim_{ x \to a } \frac{\alpha_{1}(x)}{\beta(x)} }+ \cancelto{ \ 0 }{ \lim_{ x \to a } \frac{\alpha_{2}(x)}{\beta(x)} }=0$

### Асимптотика

1. $\sin x=x+0(x)$ при $x\to \infty$
2. $\cos x=1-\frac{x^{2}}{2}+o(x^{2})$
3. $\tan x=x+o(x)$
4. $\ln(1+x)=x+o(x)$
5. $a^{x}=1+x\ln a+o(x)$
6. $e^{x}=1+x+o(x)$
7. $(1+x)^{a}=1+ax+o(x)$
8. $\mathrm{sh}\ x=x+o(x)$
9. $\mathrm{ch}\ x=1+ \frac{x^{2}}{2}+o(x^{2})$
10. $\mathrm{th}\ x=x+o(x)$
