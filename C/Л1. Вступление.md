### Код программы

```c
#include<stdio.h>
int even(int x){
	if (x % 2 != 0){
		x++;
	}
	return x;
}

int main(){
	int val = 0;
	int res = 0;
	scanf("%d", &val);	
   
    res = even(val);
    printf("Result is: %d\n", res);

    return 0;
}
```
### Таблица трассировки

**Для нечетного val:**

|               | val | res |
| ------------- | --- | --- |
| 1             | 0   | 0   |
| scanf()       | 3   | 0   |
| res=even(...) | 3   | 4   |

**Для четного val:**

|               | val | res |
| ------------- | --- | --- |
| 1             | 0   | 0   |
| scanf()       | 2   | 0   |
| res=even(...) | 2   | 2   |

Весь алгоритм должен быть покрыт тестовыми примерами.

### Пример хода решения

1. $395\to 593 \to 395 + 593 \to 988$

4. $9658 \to 8569 \to 9658 + 8569 = 18227$
5. $18227 \to 72281 \to 18227 + 72281 = 90508$
6. $90508\to 80509 \to 90508 + 80509 = 171017$
7. $171017\to 710171 \to 171017 + 710171 = 881188$ (палиндром!)
