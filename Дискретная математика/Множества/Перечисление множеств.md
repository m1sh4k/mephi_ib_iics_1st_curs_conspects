### Матрица инциденции

По дугам, какие дуги куда ведут
## Отношения
### Рефлексии
-  **Рефлексивная**
	Сама на себя
	$a\to a$
	$b\to b$

- **Иррефлексивное**
	Отрицание рефлексивности
- **Нерефлексивное**
	Есть и замкнутые и незамкнутые вершины.

### Симметрия
- **Симметричное**
	Есть есть дуга в одну, то есть и в другую сторону.
	$a\to b\implies b\to a$
- **Антисимметричное**
	Нет симметричных дуг
- **Несимметричное**
	Есть симметричные, но не все.

### Транзитивные

- **Транзитивное замыкание**
	Если есть две последующие дуги, то есть и замыкающая.
	Есть $a\to b\to c\implies$ есть $a\to c$

- **Интранзитивное**
	Когда нет НИ ОДНОЙ транзитивности.

- **Нетранзитивное**
	Есть транзитивные, но не все.
